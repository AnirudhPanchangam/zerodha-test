#Small web app for Zerodha test

To run this application ,you will need redis-server installed on your machine.
 ```bash
 sudo apt update
 sudo apt install redis-server
 ```
You will also need some python packages

 ```
 pip3 install cherrypy redis
 ```

To run the app, change to the desired directory and run :

 ```
 python3 app.py
 ```
